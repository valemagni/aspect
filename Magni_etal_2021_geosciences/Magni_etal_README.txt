Ridge jumps and mantle exhumation in back-arc basins 
V. Magni, J. Naliboff, M. Prada, C. Gaina
Geosciences, 2021

This repository contains the version ASPECT used in Magni et al., 2021 and the parameter files in the directory “Magni_etal_2021_geosciences”

Parameter files of the reference models for each scenario:
- Asymmetric margins (Fig.3): v1t8_v0t3_v1.prm
- Continental fragment formation (Fig. 4): v2t6_v0t3_v2.prm
- Intra-oceanic ridge jump (Fig. 5): v2t7_v0t3_v2.prm

All other models in this study have the same parameter files but different v1, v2, t1, and t2 values, which can be changed in line 110 (see Table 2 for the used values).

Other files:
- Example of parameter file of models with acceleration and deceleration in the transition phase:  v2t6_v202t3_v2.prm
- Initial plastic strain field: mycomp_Long1_2.5.txt
